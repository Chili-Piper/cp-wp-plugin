== Description ==

This plugin allows you to integrate your form with ChiliPiper Concierge without adding any other script.

After you have set your router up on Chili Piper, get the tenant and router slug under the deployment tab.
Edit the page or post you have your form and, on the right side, add the tenant and router information you got from ChiliPiper.


== Installation ==
This section describes how to install the plugin and get it working.

1. Upload [`concierge`](https://github.com/Chili-Piper/cp-wp-plugin/archive/refs/heads/main.zip) to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Under any post or page which contains a form, add the tenant and the router information you got after publishing you Concierge router
